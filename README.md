# MJM-AI: Plantation AI Operations Hub

A centralized dashboard for plantation management, integrating operations, finance, and machinery records with AI analysis.

## Core Modules
* **Manuring Records** - Tracking fertilizer application.
* **Field Maintenance** - Chemical and weed management logs.
* **Financial Reports** - Monthly P&L and operational costs.
* **Machinery** - Maintenance and fuel tracking.

## Tech Stack
* **Frontend:** HTML5, Tailwind CSS
* **Backend:** Supabase (Database & Storage)
* **Intelligence:** AI-powered analysis for report insights.
